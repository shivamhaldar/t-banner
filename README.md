# T_banner

## Installation on termux---
```
1. git clone https://github.com/shivamhaldar/t-banner.git
2. cd t-banner
3. chmod +x *
4. ./banner.sh

```
### One line cmd 
```
git clone https://github.com/shivamhaldar/t-banner.git;cd t-banner;chmod +x *;bash banner.sh 
```
