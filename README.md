## Installation on termux---
```
1. git clone https://github.com/shivamhaldar/t-banner.git
2. cd T_banner
3. chmod +x *
4. ./banner.sh

```
### One line cmd 
```
git clone https://github.com/shivamhaldar/t-banner.git;cd T_banner;chmod +x *;bash banner.sh 
```
